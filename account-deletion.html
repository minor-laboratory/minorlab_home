계정 삭제

1. 앱에서 직접 삭제

BookLab 앱 → 더보기 탭 → 계정 관리 → 회원 탈퇴

⚠️ 중요 안내
계정 삭제 시 모든 데이터(독서 기록, 메모, 목표 등)가 영구적으로 삭제되며 복구할 수 없습니다.

2. 이메일로 요청

아래 폼을 작성하여 계정 삭제를 요청하세요

문의사항이 있으시면 danny@minorlab.com으로 연락해 주세요.

이메일 보내기

처리 절차

  • 요청 접수 후 3일 이내에 계정 확인 이메일 발송
  • 이메일 내 삭제 확인 링크 클릭 시 계정 및 데이터 완전 삭제
  • 삭제 완료 알림 이메일 발송
